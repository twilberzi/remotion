# Scene Starter Prompts

A shared library of prompts that produced good results. Copy, tweak, and reuse.

---

## How to add an entry

Paste the prompt that worked, note what composition it produced, and drop the output file in `examples/` with the same name.

```
### [Short name]
**Prompt:** [exact or paraphrased prompt]
**Output:** [examples/FileName.jsx]
**Notes:** [anything worth remembering — what made it work, what to watch for]
```
