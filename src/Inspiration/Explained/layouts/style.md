# Layouts — Master Style Guide

## Available Layouts

### 1. Chapter Title (`/layouts/chapter_title/`)
Full screen centered. Pill label + large headline. No graphics.
Use for: chapter transitions, topic resets.

### 2. Chapter List (`/layouts/chapter_list/`)
Half split. Left: numbered list with sliding highlight pill. Right: dark panel.
Use for: frameworks, step-by-step, "ways to…" lists.

### 3. Half Split (`/layouts/half_split/`)
50/50. Left: text (headline + supporting). Right: diagram or visual.
Use for: concept + visual explanation together.

### 4. Text Callout Bottom Screen
Lower-third style. Text appears at bottom as words are spoken.
Use for: emphasis lines, quotes, key stats.

### 5. Full Bleed Text Overlay
Full screen background with large centered quote or stat.
Use for: punchy opening lines, closing statements.

### 6. Icon Grid
3–4 columns of icon tiles. Each tile: icon + label + optional badge.
Use for: lists of features, data types, roles.

### 7. Hub & Spoke Full Scene
Centered diagram filling most of the canvas. Title above or below.
Use for: system diagrams, buying committees, data relationships.

## Global Layout Rules
- Canvas: 1920×1080px
- Safe zone: 80px margin all sides
- Content max width: 1760px
- Vertical center: 540px
- Background: always use Background.png or light gradient (#EEF2FF→#F8FAFF)

## Typography Scale
- Scene headline: 56–72px, weight 800, NAVY, letter-spacing -1px
- Section label: 28–36px, weight 700, NAVY
- Body/supporting: 22–26px, weight 500, NAVY at 70% opacity
- Caption/label: 14–16px, weight 600, muted navy

## Color Usage
- Primary text: NAVY #1e2d5a
- Accent/highlight: RED #e8182e → PINK #c2185b gradient
- Background: LIGHT #f8faff
- Card background: #ffffff
- Muted text: rgba(30,45,90,0.55)

## Spacing System
- Between sections: 60–80px
- Between cards: 32–40px
- Card internal padding: 20–28px
- Icon to label gap: 10–14px
