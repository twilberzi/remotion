# Layout: Half Split (Text + Visual)

## Purpose
Use this layout when explaining a concept and showing a diagram or workflow at the same time.

## Structure
- Left side: Text (headline + 1–2 short lines)
- Right side: Visual (icons, boxes, workflow, or chart)
- 50 / 50 horizontal split

## Motion Rules
- Text fades in first
- Visual animates in second
- Arrows or connections animate last

## Spacing
- Generous padding
- Keep visuals centered in right panel
- Text max width 40–50% of screen

## When to Use
- Explainers
- Step-by-step concepts
- Framework breakdowns

## Style
- Dark background
- Light card boxes
- Brand icons only
